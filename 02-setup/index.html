インストールの前に、まず決める

Claude Code には、3つの入り方がある

課金形態が違うだけで、使えるものはほぼ同じ

従量課金

API

使った分だけ課金。試すだけなら数円〜数十円。

開発者・ヘビーユーザー向け。上限なし。

▼ 学生に勧める入口

Claude Pro

月 $20（約 3,000 円）の定額。

claude.ai アカウントがあれば即使える。まずここから。

大容量

Claude Max

月 $100 / $200。使用量の上限が大幅に拡張。

毎日ガッツリ使う人向け。最初は不要。

※ 無料プランは Claude Code 非対応（公式明記）。Pro 以上が必要。本講義では Claude Pro を前提に進めます。料金・プラン詳細は anthropic.com で最新情報を確認してください（2026/5 時点）。

Chapter 2 · インストール方法 04 / 12
インストールの前に、方法を選ぶ

インストール方法は 4つ ある

ほとんどの人は Native Install（推奨） 一択

▼ 推奨

Native

curl / PowerShell スクリプト 1 行。前提不要。

自動アップデートあり。次ページで詳細。

Mac のみ

Homebrew

brew install --cask claude-code

手動アップデート。Homebrew ユーザー向け。

Windows のみ

WinGet

winget install Anthropic.ClaudeCode

手動アップデート。

上級者向け

npm

npm install -g @anthropic-ai/claude-code

Node.js 18+ が必要。開発者向け。

※ 出典：code.claude.com/docs/en/getting-started（2026/5 時点）。Native Install は前提ソフト不要・自動アップデートあり。迷ったらこれを選ぶ。

Lecture 02 Chapter 2 · インストール方法
Chapter 3 · Native Install 05 / 12
推奨方法のコマンドを OS 別に示す

ターミナルに 1行 貼り付けるだけ

macOS / Linux / WSL
curl -fsSL https://claude.ai/install.sh | bash
Windows PowerShell
PS C:\> のプロンプト
irm https://claude.ai/install.ps1 | iex
Windows CMD
C:\> のプロンプト
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
インストール後の確認
claude --version

※ Native Install は自動アップデートあり。プロンプトが PS C:\ なら PowerShell、C:\ だけなら CMD。迷ったら PowerShell を使う。

Chapter 4 · 認証 06 / 12
インストールの次は、アカウントと紐づける

ターミナルで claude と打つだけ

初回起動時に、自動でログインを促してくれる

Claude Pro / Max の場合
claude

ブラウザが開き、claude.ai のアカウントでログイン。

認証が完了すると、ターミナルに戻って Claude との会話が始まる。

API キーを使う場合

anthropic.com でダッシュボードから API キーを発行。

初回起動時にキーの入力を求められるので、貼り付ける。

キーは安全な場所に保存する（再発行可能）。

※ ログインは一度やれば、次回からは不要。セッションが保持される。

Chapter 5 · 初起動 07 / 12
ログインが完了したら

まず、挨拶 してみる

動いているかを確認するだけでいい — 難しいことは後でいい

最初に試すこと

ターミナルで claude を起動し、こう打つ：

こんにちは。動いてますか？

返事が来たら、セットアップ完了。

役立つコマンド
/help      使えるコマンド一覧
/clear     会話をリセット
/exit      終了

スラッシュコマンドは Claude Code 専用の操作。

※ 作業ディレクトリを意識する。cd your-project で移動してから claude を起動すると、そのフォルダのファイルを Claude が認識できる。
